# 플랫폼
윈도우 데스크 톱
# 장르
3d rpg
# 차별화 요소
맵 하나를 뒤져보며 몬스터를 사냥하고 코인을 모으는것에 주력. 
레벨, 아이템의 요소는 존재하지 않으며 coin을 모으는 점수에 주력함.
# 플레이 방식
게임의 목표는 coin을 모으는 단순한 rpg이며 coin이 score이자 목표이다.
게임오버가 될때까지 몬스터를 사냥하여 coin을 모으는것이 목적이며 game over가 됐을때에 점수로 순위를 매김.
일반 몬스터와 보스 몬스터가 각각 존재하며 주는 coin을 다르게 설정할것임. 몬스터가 죽고나면 그 자리에 리스폰이 됨.
